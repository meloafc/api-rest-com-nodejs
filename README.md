## Instalação

* Clone o repositório: `git clone https://github.com/meloafc/api-rest-com-nodejs.git`
* Acesse o diretório do projeto: `cd ntask-api`
* Instale as dependências: `npm install`
* Inicie o servidor: `npm start`
